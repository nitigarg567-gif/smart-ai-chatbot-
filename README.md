## 💡 About the Project

**StudyBuddy AI** is an intelligent, all-in-one study companion chatbot designed to make learning smarter, faster, and more fun. Built for students who need a personal AI tutor available 24/7 — from doubt solving to motivation, we've got it all covered.

---

## ✨ Features

| Feature | Description |
|---|---|
| 📅 **Study Planner** | Generate personalized weekly study plans based on your subjects and exam schedule |
| 💬 **Quick Doubt Help** | Get instant, simple explanations for any concept with examples and analogies |
| 🔥 **Motivation Mode** | Receive powerful motivational messages when you're feeling low on energy |
| ⚡ **Quick Quiz** | Test your knowledge with AI-generated MCQs with instant feedback |
| 🃏 **Flashcards** | Create interactive flip flashcards for quick revision of any topic |
| 🧠 **Mind Maps** | Auto-generate visual mind maps to understand topic connections clearly |

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **AI:** Groq ai, Meta's Llama 3.3708
- **Deployment:** GitHub Pages

---

## 🏃 How to Run

1. Download `index.html`
2. Double click — opens in browser
3. Get free API key from groq.com
4. Paste key and click Launch!

---

## 🔑 Get Free API Key

1. Go to **console.groq.com**
2. Sign up with Google
3. Click **"Create API Key"**
4. Copy key starting with **gsk_**
5. Paste in app — done!
